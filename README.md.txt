# AI 私人助手（GPT‑4o‑mini 版）

这是一个基于 **GPT‑4o‑mini** 的 AI 私人助手，支持：
- 🗒 记录待办任务
- 💬 智能对话（多轮记忆）
- 💰 API 调用费用统计
- 👥 多用户独立会话（使用 session_id 区分）

---

## 1️⃣ 项目简介
本项目使用 **GPT‑4o‑mini** 作为对话模型：
- 与 GPT‑4o 性能接近，但价格更低、响应更快
- 官方定价（美元 / 百万 tokens）：
  - **输入（Prompt Tokens）**：$0.15
  - **输出（Completion Tokens）**：$0.60
- 调用方式和 GPT‑3.5 一致，仅需修改 `model` 参数为 `gpt-4o-mini`

---

## 2️⃣ 功能列表
1. **待办记录**：添加、查看任务列表  
2. **多轮记忆**：在同一个会话中保留上下文信息  
3. **费用计算**：返回本次调用的 token 统计和美元价格  
4. **多用户支持**：不同 session_id 拥有独立的记忆空间  
5. **会话重置**：手动清除某个用户的历史对话
